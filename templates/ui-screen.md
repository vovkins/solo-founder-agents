## Блоки

### 1. Header

**Компоненты:**
- `Logo` — логотип
- `Navigation` — навигационное меню
- `Avatar` — аватар пользователя

**Позиция:** верх экрана, фиксированный

---

### 2. Main Content

**Компоненты:**
- `Card` — карточка с контентом
  - `Heading` — заголовок
  - `Text` — описание
  - `Button` — действие

**Позиция:** центр экрана

---

### 3. Footer

**Компоненты:**
- `Link` — ссылки
- `Text` — копирайт

**Позиция:** низ экрана

---

## Навигация

### Откуда можно попасть на экран:
- Home → клик по ссылке
- Login → после успешной авторизации

### Куда можно перейти:
- Profile → клик по аватару
- Settings → клик по ссылке в меню
