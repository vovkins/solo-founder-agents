# Infrastructure

## Хостинг

**VPS (облачный сервер)** — доступность 24x7.

**Docker контейнеры** — CrewAI и все компоненты системы разворачиваются в контейнерах для:
- Портативности
- Изоляции
- Легкого деплоя и обновлений

---

## Хранение состояния

**Текущий подход:** Markdown и JSON файлы

- Артефакты в Markdown (PRD, ADR, test cases)
- Метаданные и конфигурация в JSON
- Git-friendly, просто для MVP

**Будущее:** PostgreSQL

- При масштабировании
- Для сложных запросов и отношений
- Migration path: JSON → PostgreSQL

---

## Интеграция с GitHub

**Personal Access Token (PAT)**

- Токен выпускается в Settings → Developer settings → Personal access tokens
- Права: repo (full control), issues, projects
- Токен хранится в .env (не в коде!)

```env
GITHUB_TOKEN=ghp_...
GITHUB_REPO=owner/repo-name
```

---

## Интерфейс для Founder

**Два канала:**

1. **CLI** — для администрирования, отладки, продвинутых операций
2. **Telegram Bot** — для повседневного взаимодействия

Архитектура аналогична OpenClaw:
- Gateway service обрабатывает входящие сообщения
- Telegram adapter для бота
- CLI adapter для командной строки

## Архитектура системы (предварительная)

```
┌─────────────────────────────────────────────┐
│                 Founder                      │
│         (CLI / Telegram Bot)                 │
└─────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────┐
│            Gateway / API Layer               │
│    (приём сообщений, маршрутизация)          │
└─────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────┐
│           CrewAI Orchestrator                │
│    (координация агентов, задачи)            │
└─────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────┐
│              Agent Crew                      │
│  PM | Analyst | Architect | Designer         │
│  Developer | QA | Tech Writer                │
└─────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────┐
│          External Integrations               │
│    GitHub API | OpenRouter LLM               │
└─────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────┐
│            Storage Layer                     │
│    Markdown/JSON files | Git repo            │
└─────────────────────────────────────────────┘
```
